Ripple and Rippay

What's Ripple? Ripple is a payment system, currency exchange and remittance network. And Rippay is a ripple gateway, a part of this system. Rippay enables users to put RMB into and take RMB out of the Ripple network.

Building Rippay

Since Ripple is a whole new thing for us, the main challenge for building Rippay is to make people understand what it is. While using Rippay, users should get information when they need it

Rippay is designed to be a simple product in the beginning, which only contains several basic features. What we did was to show guides and help though the users flows.
